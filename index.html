Sports Management System

Sports Management System is a role-based web application featuring four dedicated views — Admin, Player, Coach, and Captain — designed to digitalize and streamline sports operations within an organization. The core feature of the system is a structured budget requesting module, where coaches and captains can submit funding requests for tournaments, equipment, and events, and administrators can review, approve, or reject them through a centralized dashboard. In addition to budget management, the system includes player registration, team creation and management, match scheduling, performance tracking, and real-time status updates. With secure role-based access control, each user has access to functionalities tailored to their responsibilities, ensuring organized, transparent, and efficient sports administration.

Personalized Fitness Recommendation System

A terminal-based health and fitness recommendation system that uses an N-ary tree structure to simulate an interactive decision-making quiz. Each internal node represents a question, each edge represents a user’s selected choice, and each leaf node produces a personalized fitness plan. The system traverses from root to leaf based on user input, ensuring structured, hierarchical recommendations through a command-line interface.
